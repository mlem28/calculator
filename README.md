# Калькулятор

Простой веб-калькулятор для выполнения базовых арифметических операций.

## Стек технологий
- HTML
- CSS  
- JavaScript

## Инструкция запуска
1. Откройте файл `calculator.html` в любом браузере
2. Или перейдите по ссылке: https://mlem28.github.io/calculator/
